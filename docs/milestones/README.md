# Milestones

This folder holds the project roadmap as milestone specs. Each milestone file includes:
- goal + done definition
- scope (in/out)
- step-by-step plan with checkpoints
- acceptance criteria (copyable into a GitHub Issue)

## Big picture
- **Milestone 1–2:** Core backend kernel (Domain + Application) — framework-free truth engine.
- **Milestone 3–4:** Adapters — HTTP API + PostgreSQL persistence.
- **Milestone 5:** Ingestion + “Peace” calculations.
- **Milestone 6:** Calm UI (React MVP).
- **Milestone 7:** Safe data management (account/transaction CRUD).
- **Milestone 8:** Recurring transactions + templates.
- **Milestone 9:** Goals + planning flows.
- **Milestone 10:** Richer imports + preview/rules.
- **Milestone 11:** Russian FIRE engine + backend refactor.
- **Milestone 12:** Closed beta launch on Yandex Cloud.
